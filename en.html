Steps

Green = actions
1

Install Equalizer APO

  • Download: https://sourceforge.net/projects/equalizerapo/
  • Right click ‚Üí Run as administrator
  • Select only your real headset/speakers (USB Headset / Realtek Speakers)
  • Do not select monitor / HDMI / NVIDIA / Sonar / virtual devices
  • Finish ‚Üí Restart PC
2

Install Peace

  • Download: https://sourceforge.net/projects/peace-equalizer-apo-extension/
  • Choose Full Interface
  • Open Peace
3

Install ReaPlugs

  • Download: https://www.reaper.fm/reaplugs/
  • Install ReaPlugs VST 64-bit with defaults
4

Open the VST Editor and set the chain

  • Peace ‚Üí Commands ‚Üí Use Editor (VST)
  • Tab: Before Peace EQ
  • If you see config.txt, delete that tab

Recommended chain: Preamp ‚Üí ReaComp ‚Üí Graphic EQ

5

GAIN before the compressor

  • Press + ‚Üí Basic Filters ‚Üí Preamp
  • Set -7.00 dB
6

Add ReaComp (VST)

  • Press + ‚Üí Plugins ‚Üí VST plugin
  • Select reacomp-standalone.dll (ReaPlugs)
  • Click Open panel
7

Save

  • Ctrl + S in the Configuration Editor
